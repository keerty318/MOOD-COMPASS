# 🧭 MoodCompass

MoodCompass is a simple and interactive mood journaling web application that helps users reflect on their emotions. Users can choose their current mood, write journal entries, practice breathing exercises, view their mood history, and explore self-care tips—all in a clean and calming interface.

---

## ✨ Features

- 😊 Select your current mood
- 📝 Write and save journal entries
- 📊 View mood history and recent timeline
- 🌟 See your most frequent mood
- 🫁 Guided breathing exercise
- 💡 Wellness and self-care tips
- 🎨 Responsive and aesthetically pleasing UI
- 💾 Uses Local Storage to save user data

---

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `HTML/index.html` in your browser.
4. Start tracking your mood.
